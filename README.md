# Challenge Docker Nginx como Proxy Reverso e NodeJS

## Como executar

Faça o clone do projeto:
```bash
  git clone
```

Suba o container:
```bash
  docker-compose up -d
```

Após isso, basta acessar na url: http://localhost:8080;
