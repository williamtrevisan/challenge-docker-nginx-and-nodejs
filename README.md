# Challenge Docker Nginx como Proxy Reverso e NodeJS

## Como executar

Faça o clone do projeto:
```bash
  git clone
```

Acesse a pasta do projeto e instale as depedências do mesmo:
```bash
  npm install
```

Suba o container:
```bash
  docker-compose up -d
```

Após isso, acessar na url: localhost:8080;
